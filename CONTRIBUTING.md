Found an issue with the [live demo](https://postpilot-io.vercel.app)?
[Open an issue](https://github.com/mzstudiospk/PostPilot-showcase/issues) with:

- Clear description of the bug
- Steps to reproduce
- Browser/device information
- Screenshots if applicable

## 🎨 About PostPilot

PostPilot is an AI-powered social media post generator built with:

- **Next.js 16** + **React 19** + **TypeScript 5**
- **Supabase** for authentication and database
- **Google Gemini AI** for content generation
- **Tailwind CSS 4** + **Framer Motion** for design

Try it live at: [postpilot-io.vercel.app](https://postpilot-io.vercel.app)
